Jak Dodać Swój Projekt

Aby dodać swoją bibliotekę lub plugin do Bytes.io, postępuj według poniższych kroków. Proces jest prosty i otwarty dla społeczności.

  1. Przygotuj Projekt: Upewnij się, że Twój kod jest w repozytorium GitHub, z jasnym README.md opisującym instalację, użycie i zależności.
  2. Zgłoś Projekt: Otwórz nowe Issue na GitHub Issues z tytułem "Dodaj Pakiet: [Nazwa Pakietu]".
  3. Dodaj Szczegóły: W Issue podaj: nazwę pakietu, wersję, link do repo, krótki opis, wymagania i przykłady użycia.
  4. Dyskusja: Jeśli chcesz omówić szczegóły, użyj GitHub Discussions.
  5. Przegląd: Nasz zespół sprawdzi projekt pod kątem jakości i zgodności z Hacker Lang. Po akceptacji, pakiet pojawi się w repozytorium.

Pamiętaj: Pakiety muszą być open-source (licencja MIT lub podobna) i kompatybilne z Hacker Lang. Unikaj duplikatów istniejących rozwiązań.

Dołącz do ekosystemu i pomóż rozwijać Hacker Lang!
