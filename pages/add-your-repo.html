Jak Dodać Swój Projekt

Aby dodać swoją bibliotekę lub plugin do Bytes.io, postępuj według poniższych kroków. Proces jest prosty i otwarty dla społeczności, zapewniając wysoką jakość wszystkich pakietów.

  1. Przygotuj Projekt: Upewnij się, że Twój kod jest w repozytorium GitHub, z jasnym README.md opisującym instalację, użycie, zależności i przykłady kodu.
  2. Zgłoś Projekt: Otwórz nowe Issue na GitHub Issues z tytułem "Dodaj Pakiet: [Nazwa Pakietu]".
  3. Dodaj Szczegóły: W Issue podaj: nazwę pakietu, wersję, link do repo, krótki opis, wymagania, przykłady użycia oraz ewentualne testy.
  4. Dyskusja: Jeśli chcesz omówić szczegóły lub uzyskać feedback, użyj GitHub Discussions.
  5. Przegląd: Nasz zespół sprawdzi projekt pod kątem jakości, bezpieczeństwa i zgodności z Hacker Lang. Po akceptacji, pakiet pojawi się w repozytorium w ciągu 48 godzin.
  6. Aktualizacje: Po dodaniu, możesz zgłaszać aktualizacje poprzez Pull Request lub nowe Issue.

Pamiętaj: Pakiety muszą być open-source (preferowana licencja MIT, GPL lub Apache) i kompatybilne z najnowszą wersją Hacker Lang. Unikaj duplikatów istniejących rozwiązań – sprawdź repozytorium najpierw.

Dołącz do ekosystemu i pomóż rozwijać Hacker Lang! Jeśli masz pytania, skontaktuj się poprzez Discussions.
